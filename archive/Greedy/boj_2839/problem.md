# BOJ 2839 - 설탕 배달

- 링크: https://www.acmicpc.net/problem/2839
- 난이도: 브론즈 1
- 분류: 그리디, 수학
- 풀이 일자: 2025-11-23

## 문제 설명

상근이는 3kg 봉지와 5kg 봉지만을 사용해 정확히 `N`kg의 설탕을 배달해야 한다. 더 적은 개수의 봉지를 사용하는 것이 목표이며, 정확히 나눌 수 없다면 `-1`을 출력해야 한다.

## 입력

하나의 정수 `N (3 ≤ N ≤ 5000)`이 주어진다.

## 출력

정확히 `N`kg을 만들 수 있는 최소 봉지 개수를 출력한다. 만들 수 없으면 `-1`을 출력한다.

## 예제

| 입력 | 출력 |
|------|------|
| 18   | 4    |
| 4    | -1   |
| 6    | 2    |
| 9    | 3    |
| 11   | 3    |
| 3    | 1    |
| 5    | 1    |
