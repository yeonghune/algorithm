# BOJ 11725 - 트리의 부모 찾기

- 링크: https://www.acmicpc.net/problem/11725
- 난이도: 실버 2
- 분류: 트리, 그래프, BFS/DFS
- 정리 일자: 2025-11-30

## 문제 설명

루트가 1인 트리가 주어졌을 때, 각 노드의 부모 노드를 구하는 문제이다.  
노드 1은 루트이므로 부모가 없고, 2번 노드부터 N번 노드까지 각각의 부모를 한 줄에 하나씩 출력해야 한다.

## 입력

1. 첫 줄에 노드의 개수 `N (2 ≤ N ≤ 100,000)`이 주어진다.  
2. 다음 `N-1`줄에 두 정수 `a b`가 주어지며, 이는 `a`와 `b`가 서로 연결되어 있다는 의미이다.

## 출력

2번 노드부터 N번 노드까지 각 노드의 부모 노드 번호를 한 줄에 하나씩 출력한다.

## 예시

| 입력 | 출력 |
|------|------|
| `7`<br>`1 6`<br>`6 3`<br>`3 5`<br>`4 1`<br>`2 4`<br>`4 7` | `4`<br>`6`<br>`1`<br>`3`<br>`1`<br>`4` |

## 메모

- 트리는 사이클이 없는 연결 그래프이므로, 임의의 노드(여기서는 1번)에서 BFS/DFS를 한 번만 돌면 모든 노드의 부모를 결정할 수 있다.
- 인접 리스트를 만든 뒤, 루트(1)에서 시작해 방문하지 않은 노드의 부모를 현재 노드로 기록하면서 탐색한다.
- 시간 복잡도는 간선 수가 `N-1`이므로 `O(N)`이다.
